 # RPA: Automatización de Registro de Datos Multiplataforma (Excel a Web)

## Descripción del Proyecto
Este bot fue diseñado para resolver un cuello de botella operativo crítico: el registro manual masivo de datos. La automatización extrae información desde archivos de Excel y realiza el registro sincronizado en dos plataformas web distintas, garantizando la integridad de los datos y optimizando los tiempos de respuesta mediante procesamiento en paralelo.

## Impacto del Proyecto (Resultados Reales)
* **Ahorro de Tiempo:** Reducción drástica de **6 horas diarias a solo minutos**.
* **Precisión:** Eliminación del 100% de los errores humanos por digitación.
* **Eficiencia:** Liberación de carga operativa para que el equipo se enfoque en tareas de análisis y estrategia.

## Tecnologías y Herramientas
* **UiPath Studio** (Desarrollo del flujo de trabajo).
* **Web Automation:** Uso de selectores dinámicos y robustos.
* **Data Manipulation:** Manejo avanzado de Excel y DataTables.
* **Parallel Processing:** Ejecución optimizada para mejorar el rendimiento.
* **Error Handling:** Validaciones de datos y manejo de excepciones.

## Funcionalidades Clave
1.  **Lectura Inteligente:** Extracción estructurada de más de 1500 registros diarios.
2.  **Sincronización Web:** Registro automatizado en dos portales web de forma simultánea.
3.  **Validación:** Sistema de verificación para asegurar que cada campo se complete correctamente.
