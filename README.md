# data-fling-automation-uipath
Herramienta:UiPath
Categoría: Automatización de procesos / RPA
Problema:
El proceso manual de revisar facturas en PDF y registrar su información en Excel tomaba entre 1 y 2 horas, con alto riesgo de errores por digitación.
Solución:
Bot desarrollado en UiPath que recorre una carpeta de facturas en formato PDF, extrae los campos clave (número de factura, fecha de emisión y valor total) mediante expresiones regulares, y consolida toda la información en un archivo Excel estructurado.
Resultado:
Reducción del tiempo de procesamiento de 2 horas a menos de 1 minuto, eliminando errores manuales y liberando tiempo para tareas de mayor valor.
Habilidades aplicadas:
UiPath Studio · Leer texto PDF · Regex · Manejo de DataTable · Escritura en Excel
